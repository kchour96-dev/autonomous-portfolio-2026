Who Built This

My name is Kchour, a self-taught developer from Phnom Penh, Cambodia. I built this dashboard to learn how to combine multiple AI systems into one autonomous pipeline.

I started with just a domain name and a basic idea — use AI to automatically track crypto security threats and market signals, and publish them without any manual work. This site is the result of that experiment.

What This Site Does

Autonomous Lab 2026 is a real-time intelligence dashboard that monitors crypto security threats, Web3 vulnerabilities, and market signals. It updates automatically every 2 hours using a multi-agent AI pipeline.

  • › Pulls live headlines from 6 RSS news sources
  • › Researches deeper context using Tavily Search
  • › Generates expert analysis using Google Gemini AI
  • › Adds contrarian critique using Groq / Llama 3.3
  • › Shows live crypto prices from CoinGecko
  • › Sends Telegram notifications on each update

Why I Built This

I wanted to build something that runs by itself — a system that wakes up every 2 hours, reads the latest crypto news, thinks about what it means, and publishes a clear report for anyone to read. No manual work. No cost. Fully autonomous.

This is also my portfolio project to show what I can build with AI APIs, GitHub Actions, and web development — all for free.
